# Product: dinrplan

dinrplan is a personal meal planner and recipe database web application. Users can:

- View a weekly calendar grid showing planned meals per day
- Add, edit, and delete meals and recipes
- Drag and drop meals between days
- Browse and manage a recipe list
- Upload meal and day data via JSON import
- Control how many weeks are shown and how far back/forward the calendar starts

The app is multi-user — each user's meals, days, and preferences are isolated. Authentication is required to view or modify personal data; unauthenticated users see an empty planner.

Deployed at [dinrplan.fly.dev](https://dinrplan.fly.dev) via Fly.io with automated deployment on passing CI.
